How to Use

Step 1: Open the Console

Access your browser's developer tools (Ctrl + Shift + I or Cmd + Option + I on Mac)

Step 2: Paste in the ACET Code

Visit our GitHub repository to find the ACETauto.js code
Paste it into your browser's console

Step 3: Execute Your Code

Enter your HTML, CSS, or JavaScript code in the textarea
Click "Run" to execute or "Fetch" for external data
